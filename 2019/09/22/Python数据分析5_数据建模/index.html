机器学习

分类模型

  • KNN
    • KD-Tree
  • 朴素贝叶斯
    • 生成模型：通过求输入与输出的联合概率分布，再求解类别归类的概率；
    • 判别模型：不通过联合概率分布，直接可以获得输出对应最大分类的概率。
  • 决策树
    • 信息增益-ID3
    • 信息增益率-C4.5
    • Gini系数-CART
    • 几个问题：
      • 连续值的切分
      • 规则用尽
      • 过拟合
  • 支持向量机
    • 几个问题：
      • 少部分异常值：引入松弛变量
      • 样本不平衡：对不同标注赋予不同的权值
      • 多类别分类：One-Other；One-One。
    • 核方法：先在低维计算，再进行扩维
      • 线性核函数
      • 多项式核函数
      • 高斯径向基（RBF）核函数
  • 集成方法：组合多个模型，以获得更好地效果
    • bagging：训练多个弱学习器模型（相互独立），最后投票或求均值得出结果。如随机森林。
    • boosting：弱学习器之间有关联，最后结果是所有弱学习器结果的加权叠加。如Adaboost、GBDT、XGBoost等。
      • 精度高，且灵活可调
      • 几乎不用担心过拟合
      • 简化特征工程流程
  • Logistic 回归
  • 人工神经网络

回归模型

  • 线性回归
    • 正则化：岭回归、Lasso
  • 回归树、提升树-GBDT（拟合前一个学习器的差分值）
    • XGBoost：包含高阶导数、支持并行计算
  • 支持向量机回归
  • 集成方法回归
  • Logistic 回归
  • 人工神经网络
    • 激活函数
    • 反向传播
    • 梯度下降
    • 正则化、dropout

聚类模型

  • 基于切割的Kmeans算法
    • 几个问题：
      • 初始质心位置可能影响最终聚类结果
      • 个别离群值会影响整体聚类效果
      • 必须要指定K：借助其它衡量因子进行辅助（如轮廓系数）
  • 基于层次的聚类
    • 簇间的距离衡量：最短距离、最长距离、平均距离、Ward
    • 特点：
      • 聚类灵活
      • 计算复杂度高，离群点影响大
  • 基于密度的DBSCAN：找到密度相连对象的最大集合
    • E邻域：给定对象半径为E内的区域称为该对象的E邻域。
    • 核心对象：如果给定对象E邻域内的样本点数大于等于MinPts，则称该对象为核心对象。
    • 直接密度可达：对于样本集合D，如果样本点q在p的E 邻域内，并且p为核心对象，那么对象q从对象p直接密度可达。
    • 密度可达
    • 密度相连
    • 特点：
      • 离群点不敏感
      • 需要KD-Tree等数据结构辅助
  • 基于图的Split
    • 与基于层次聚类思路相反，为自顶向下
    • 图建立方式和分裂方式非常灵活

关联规则

  • 反映一个事物与其他事物之间的相互依存性和关联性（啤酒尿布）
  • 概念：
    • 项目：一个字段，如一次交易中的一个物品
    • 事务：某个客户在一次交易中，发生的所有项目的集合
    • 项集：包含若干个项目的集合（一次事务中的）
    • 频繁项集：某个项集的支持度大于设定的阈值
    • 支持度：某一项集在总项集中出现的概率
    • 置信度：在先决条件X发生的条件下，由关联规则退出Y的概率
    • 提升度：表示含有X的条件下同时含有Y的概率，与无论含不含X含有Y的概率之比
  • Apriori算法
