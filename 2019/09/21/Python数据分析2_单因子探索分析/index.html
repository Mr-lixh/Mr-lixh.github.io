单因子探索分析

理论知识

  • 集中趋势：均值、中位数与分位数、众数
    • 四分位数计算：(n+1)*0.25，0.5，0.75
  • 离中趋势：标准差、方差
  • 数据分布：偏态与峰态、正态分布与三大分布
    • 偏态：以平均值与中位数之差对标准差之比率来衡量数据分布的偏斜程度
    • 峰态：峰态通常是与标准正态分布相比较而言的。如果一组数据服从标准正态分布，则峰态系数的值等于0；若峰态系数的值明显不等于0，则表明分布比正态分布更平或更尖，通常称为平峰分布或尖峰分布。表明了数据分布的集中程度
    • 三大分布：卡方分布、t 分布和 F 分布
  • 抽样理论：抽样误差与抽样精度

数据分类

  • 定类（类别）：根据事物离散、无差别属性进行的分类。类别间是无差距的
  • 定序（顺序）：可以界定数据的大小，但不能测定差值。如收入的低、中、高
  • 定距（间隔）：可以界定数据大小的同时，可测定差值，但无绝对零点。即乘法、除法、比率没有意义
  • 定比（比率）：可以界定数据大小，可测定差值，有绝对零点。如身高、体重、体积等

单属性分析

  • 异常值分析：离散异常值、连续异常值、常识异常值

  • 对比分析：绝对数和相对数，时间、空间、理论维度比较
    • 相对数：将几个有联系的指标进行联合，构成新的数。
      • 结构相对数：部分与整体相比。如合格率
      • 比例相对数：整体内的不同部分进行比较。如不同产业间的比较
      • 比较相对数：同一时空下的相似或同质的指标进行对比。如不同时期下同一商品的价格，不同电商的待遇水平比较等
      • 动态相对数：含有时间概念。如速度、增速
      • 强度相对数：性质不同但有相互联系的属性进行联合。如人均、密度
    • 时间维度：如同比、环比；空间维度；经验与计划
  • 结构分析：各组成部分的分布与规律
    • 静态结构分析（饼状图）
    • 动态结构分析：以时间为轴，分析变化的动态趋势（折线图）
  • 分布分析：数据分布频率的显式分析
    • 直接获得概率分布
    • 判断是不是正态分布
    • 极大似然
