多因子探索分析

理论知识

  • 假设检验与方差检验
    • 假设检验：根据一定的假设条件，从样本推断总体或者样本与样本之间关系的一种方法。
    • 假设检验的步骤：
      1. 建立原假设H0（包含等号），H0的反命题为H1，也叫备择假设；
      2. 选择检验统计量。即根据数据的均值、标准差等性质构造一个转换函数，使数据符合一个已知的分布（正态分布等）；
      3. 根据显著性水平（一般为0.05），确定拒绝域；
      4. 计算 p 值或样本统计值，做出判断。
    • 假设检验的方法：
      • 卡方检验：用于检验两个因素之间有没有比较强的联系
      • 方差检验（f 检验）
  • 相关系数：皮尔逊、斯皮尔曼
    • Pearson 相关系数：
    • Spearman 相关系数：更适用于相对比较
  • 回归：线性回归
    • 回归：确定两种或两种以上变量间相互依赖的定量关系的一种统计分析方法。
    • 关键指标：决定系数、残差不相关（DW 检验）
  • PCA 与奇异值分解
    • PCA 一般步骤：
      1. 求特征协方差矩阵；
      2. 求协方差的特征值和特征向量；
      3. 将特征值按照从大到小的顺序排序，选择其中最大的 k 个；
      4. 将样本点投影到选取的特征向量上。
    • SVD：特征矩阵 A 分解为 mm 的酉矩阵，mn 半正定矩阵（奇异值阵），n*n 酉阵转置 V。
